Shahana Murugesan - Website package
----------------------------------
Files:
  - index.html (home)
  - about.html
  - portfolio.html
  - contact.html
  - privacy.html
  - sitemap.xml
  - robots.txt
  - assets/styles.css
  - assets/logo.svg
  - assets/img1.jpg, img2.jpg, img3.jpg (placeholders)

How to use:
  1. Download the ZIP and extract.
  2. Replace 'yourdomain.com' with your actual domain in files (search & replace).
  3. Replace GA Measurement ID 'G-XXXXXXXXXX' in index.html with your GA4 ID.
  4. Replace placeholder images in assets/ with your actual images.
  5. Host on Netlify / Vercel / GitHub Pages by pushing this folder to a repository or uploading the files.

Accessibility & SEO:
  - Skip link, semantic HTML, form labels and focus styles included.
  - Sitemap and robots files included. Submit sitemap to Google Search Console.
  - JSON-LD Organization schema and meta tags included.

Need additional features? I can:
  - Hook contact form to a serverless endpoint (Netlify/Vercel).
  - Add cookie-consent and GDPR handling for analytics.
  - Convert to React app or add blog system.
